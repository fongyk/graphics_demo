## 计算机图形学演示程序

本目录包含计算机图形学作业两个版本的工程，分别为：

1. graphics_final_mingw
2. graphics_final_linux

每个子目录均包含代码源文件和可执行文件。其中vs版本保留了工程文件，linux版本保留了Makefile。

由于没有依赖额外的库，因此不需要额外的dll来运行程序，但是可能需要msvcr.dll，msvcp.dll等。所以不保证附带的可执行文件能够正常运行，使用时最好再重新打开编译一下。
两个版本实现的功能是完全一样的，但是依赖于平台，代码有几处不同。

依赖库：[CImg](http://cimg.eu/)
